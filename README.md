# Urban Safety App

愛知県の都市安全リスクを確認する React Native / Expo アプリです。災害警戒アプリに近い画面構成で、過去一定時間のリスク、公式信号、AIまたはルール型の行動提案を表示します。

## 起動

```bash
npm install
npm start
```

`App.js` の `BACKEND_URL` は、自分のPCのローカルIPアドレスに合わせて変更してください。

## 表示内容

- 現在の重点リスク
- 高・中・低のリスク件数
- SNS・気象・交通の寄与値
- 公式信号数と更新時刻
- 気象庁XMLから保存したLive公式件数
- 過去時間帯ごとのリスク推移
- カテゴリ別のリスク件数
- 公式情報の手動更新ボタン
- 詳細画面でのAI評価と行動提案

## 注意

表示内容は模擬データに基づく参考情報です。実際の避難・移動判断では、自治体、気象庁、鉄道会社などの公式情報を確認する前提です。
